# Number Guessing Game

## Description

A simple command-line number guessing game where players try to guess a randomly generated number within a specified range.

## Features

- Random number generation
- User input validation
- Attempt counter
- Hints (higher/lower feedback)
- Replay functionality

## Project Code

[GitHub Repository Link](https://github.com/PrashantChauhan001/node-projects)

## How to Install

1. Clone the repository:

```bash
git clone https://github.com/PrashantChauhan001/node-projects
cd GuessingGame
```

## How to Run

```bash
node index.js
```

Or use npm script:

```bash
npm start
```

## How to Play

1. Run the application
2. Enter a number when prompted
3. Receive feedback if your guess is too high or too low
4. Continue guessing until you find the correct number
5. View your total attempts
6. Choose to play again or exit

## Requirements

- Node.js (v12 or higher)
